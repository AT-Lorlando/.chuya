#### **Système & Environnement**
* **zsh** : Shell alternatif à Bash, plus puissant et conçu pour l'usage interactif.
* **ohmyzsh** : Framework pour gérer la configuration de Zsh (thèmes, plugins, alias).
* **timeshift** : Outil de sauvegarde système par "snapshots" (points de restauration).
* **hyprlock** : Écran de verrouillage (lockscreen) simple et esthétique pour Hyprland.
* **nm-applet** : Icône de la barre des tâches pour gérer les connexions réseau (Wi-Fi/Ethernet).
* **nwg-look** : Interface graphique pour personnaliser les thèmes GTK, icônes et curseurs sous Wayland.
* **greetd** : Login manager pour Wayland.
* **tuigreet** : Login manager pour Wayland.

#### **Interface & Navigation**
* **dunst** : Démon de notification léger et hautement personnalisable (gère l'affichage visuel des pop-ups).
* **waybar** : Barre d'état (taskbar) hautement personnalisable pour Wayland.
* **kitty** : Émulateur de terminal moderne, rapide (accéléré par GPU) et scriptable.
* **rofi** : Lanceur d'applications et changeur de fenêtres polyvalent (menu démarrer rapide).
* **wal/pywall** : Wallpaper manager.
* **wl-clipboard cliphist** : Gestionnaire de presse-papiers.

#### **Outils Terminal (CLI / TUI)**
* **atuin** : Remplace l'historique du shell par une base de données synchronisable et cherchable (`Ctrl+R`).
* **eza** : Remplaçant moderne de la commande `ls` avec couleurs, icônes et métadonnées Git.
* **lazygit** : Interface graphique dans le terminal pour gérer Git visuellement et rapidement.
* **lazydocker** : Interface graphique dans le terminal pour gérer conteneurs et images Docker.
* **zgrep** : Commande pour rechercher du texte à l'intérieur de fichiers compressés (.gz).
* **egrep** : Version de `grep` supportant les expressions régulières étendues (alias de `grep -E`).
* **notify-send** : Commande (du paquet `libnotify`) pour envoyer une demande de notification à Dunst.
* **ncdu/gdu** : Interface graphique dans le terminal pour naviguer dans les fichiers.

#### **Gestionnaires de Fichiers**
* **yazi** : Explorateur de fichiers terminal ultra-rapide (écrit en Rust) avec prévisualisation d'images.
* **ranger** : Explorateur de fichiers terminal classique écrit en Python avec navigation type Vim.
* **superfile** : Explorateur de fichiers terminal très moderne, esthétique et dense en informations.
* **thunar** : Explorateur de fichiers graphique léger et rapide (natif de l'environnement XFCE).

#### **Cyber**
* **exegol** : Environnement de cyber complet et pré-configuré basé sur des conteneurs Docker.
